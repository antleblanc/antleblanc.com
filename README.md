## Install

```sh
$ cd ~/Sites
$ git clone https://github.com/antleblanc/antleblanc.me
$ cd antleblanc.me
$ npm install
```

## Usage

#### Developement

```sh
$ npm start
```

#### Build

```sh
$ npm run build
```

#### Test

```sh
$ npm test
```

#### Deployment

```sh
$ npm run publish
```
